# WeReadExporter-Tampermonkey

**微信读书导出工具（WeRead Exporter for Tampermonkey）**

一个基于 **Tampermonkey / Violentmonkey** 的脚本，用于在 **微信读书（WeRead）网页版**中导出书籍内容。

支持导出：

* **章节内容**
* **整本书全文**
* **Markdown / HTML / Text（TXT）格式**

适用于本地保存、归档、学习整理、二次编辑。
